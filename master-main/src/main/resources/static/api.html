云墨笔记 API 接口文档

基础信息

  • 基础URL: http://localhost:8080/api
  • 认证方式: JWT Token（请求头 Authorization: Bearer {token}）

认证接口

POST /auth/register

注册

请求体 (JSON):
{ "username": "用户名", "password": "密码" }
响应:
// 成功 { "accessToken": "JWT token", "username": "用户名" } // 失败 { "error": "错误信息" }
POST /auth/login

登录

请求体 (JSON):
{ "username": "用户名", "password": "密码" }
响应:
// 成功 { "accessToken": "JWT token", "username": "用户名" } // 失败 { "error": "Bad credentials" }
GET /auth/token/validate

验证token并查询有效期

请求头:
  • Authorization: Bearer {token}
响应:
// 成功 { "username": "用户名", "tokenExpiration": "24 hours", "status": "valid" } // 失败 { "error": "Invalid token" }

文档相关接口

POST /documents/upload

上传文档

参数:
  • file (MultipartFile): 要上传的文件
响应:
// 成功 { "data": { "id": 1, "title": "文档标题", "originalName": "原始文件名", "filePath": "文件路径", "fileType": "文件类型", "fileSize": 1024, "converted": true, "mdContent": "Markdown内容", "createdAt": "2024-01-01T00:00:00", "updatedAt": "2024-01-01T00:00:00" }, "status": 201, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" } // 失败 { "error": "错误信息", "status": 400, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" }
GET /documents

获取文档列表

参数: 无
响应:
// 成功 { "data": [ { "id": 1, "title": "文档标题", "originalName": "原始文件名", "filePath": "文件路径", "fileType": "文件类型", "fileSize": 1024, "converted": true, "mdContent": "Markdown内容", "createdAt": "2024-01-01T00:00:00", "updatedAt": "2024-01-01T00:00:00" } ], "status": 200, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" } // 失败 { "error": "错误信息", "status": 400, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" }
GET /documents/{id}

获取文档详情

参数:
  • id (路径参数): 文档ID
响应:
// 成功 { "data": { "id": 1, "title": "文档标题", "originalName": "原始文件名", "filePath": "文件路径", "fileType": "文件类型", "fileSize": 1024, "converted": true, "mdContent": "Markdown内容", "createdAt": "2024-01-01T00:00:00", "updatedAt": "2024-01-01T00:00:00" }, "status": 200, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" } // 失败 { "error": "错误信息", "status": 400, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" }
GET /documents/{id}/download

下载文档

参数:
  • id (路径参数): 文档ID
响应:
  • 成功: 文件流（Content-Disposition 头包含编码后的文件名）
  • 失败: HTTP 状态码 + 错误信息
DELETE /documents/{id}

删除文档

参数:
  • id (路径参数): 文档ID
响应:
// 成功 { "data": { "message": "Document deleted successfully" }, "status": 200, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" } // 失败 { "error": "错误信息", "status": 400, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" }
PUT /documents/{id}

更新文档

参数:
  • id (路径参数): 文档ID
  • 请求体 (JSON):
    { "title": "文档标题", "mdContent": "Markdown内容" }
响应:
// 成功 { "data": { "id": 1, "title": "更新后的标题", "originalName": "原始文件名", "filePath": "文件路径", "fileType": "文件类型", "fileSize": 1024, "converted": true, "mdContent": "更新后的Markdown内容", "createdAt": "2024-01-01T00:00:00", "updatedAt": "2024-01-01T00:00:00" }, "status": 200, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" } // 失败 { "error": "错误信息", "status": 400, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" }

统一错误响应格式

{ "error": "错误信息", "status": 400, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" }

统一成功响应格式

{ "data": "响应数据", "status": 200, "timestamp": "2024-01-01T00:00:00", "traceId": "xxxx" }
